This repo contains code required to implement a 2D Pix2Pix GAN for reconstruction of 12-lead ECGs using only 3 leads. The PTBDB, SPHADB, INCART and PTBXL databases must be downloaded first. The code will clean and prepare the data for training and evaluation. 

Reference: A. Mallick et al., "AI-Based 3-Lead to 12-Lead ECG Reconstruction: Towards Smartphone-Based Public Healthcare," 2024 IEEE International Conference on E-health Networking, Application & Services (HealthCom), Nara, Japan, 2024, pp. 1-6, doi: 10.1109/HealthCom60970.2024.10880752.
